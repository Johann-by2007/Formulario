Formulario Login Validado (HTML, CSS, JS)

Este proyecto implementa un formulario de inicio de sesión usando HTML, CSS y JavaScript puro, enfocado en validaciones del lado del cliente e interacción dinámica con el usuario.

Funcionalidades

Validación de usuario (mínimo 3 caracteres, solo letras, números, guiones y puntos).

Validación manual de correo electrónico con JavaScript.

Mostrar/Ocultar contraseña.

Contador de caracteres en tiempo real para la contraseña.

Evaluación de fortaleza de contraseña.

Mensajes visuales en la página (sin uso de alert()).

Limpieza automática del formulario tras envío correcto.

Control de intentos fallidos y bloqueo del formulario durante 30 segundos después de 3 errores.

Tecnologías usadas

HTML5

CSS3

JavaScript (Vanilla)

Objetivo

Practicar manipulación del DOM, validaciones de formularios y manejo de eventos en JavaScript sin utilizar librerías externas.
